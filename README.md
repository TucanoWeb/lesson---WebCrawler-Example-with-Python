# ToDo List by TucanoWeb with ReactJS

Project made for one of my students, using ReactJS and TS.



# Get Starter



## Install Dependencies

In the root of the project, run the command:

```bash
npm install
```



## Run Project

Yet in the root of the project, run the command:

```bash
npm start
```
